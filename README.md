# Java Learning and Practice

## Overview
This repository is created to practice Core Java concepts and improve
problem-solving skills through hands-on coding.

It contains beginner-level Java programs focused on building a strong
foundation in Java programming.

## Beginner Java Programs
The beginner package includes Java programs for:

- Even or Odd Check
- Prime Number Check
- Palindrome Number
- Reverse a Number
- Count Digits in a Number
- Largest of Three Numbers
- Multiplication Table
- Sum of Natural Numbers
- Factorial of a Number
